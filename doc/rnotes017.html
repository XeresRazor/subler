Subler 0.17 requires OS X 10.6 or later

What's New in 0.17
  • Forced subtitles tracks. A new options to set a track as "forced". iDevices will automatically display the right forced track for a given language.
  • Ocr trained data. Subler will now check ~/Library/Application Support/Subler/tessdata for Tesseract traineddata files.
  • Fixed an issue with the rating tag.
